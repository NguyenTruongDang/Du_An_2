Dựa vào các trang báo online nổi tiếng hiện nay như vnexpress, baomoi, zingnews, kenh14,...
Thì nhóm em đã liệt kết các chức năng liên quan về :
- SITE :
 + Phân loại tin
 + Update views
 + Search
 + Phân trang
 + Comment
 + Đăng nhập, Đăng kí (API Facebook, Google)
 + Lưu tin tức
 + RSS
 + SEO :
   - Tạo link thanh thiện htaccess
   - Thẻ META SEO title, keywords, Desc
   - Tags
   - sitemap


- PROFILE :
 + Gồm các chức năng UPDATE, DELETE, SELECT
 + Đăng bài:
   - Đăng bài viết
   - Xem Tin của mình đã đăng được duyệt
   - Xem tin của mình đã đăng chưa duyệt
   - Tính điểm tặng quà 
 + Xem nhật kí hoạt động:
   - Xem hoạt động comment
   - Xem Tin đã lưu


- ADMIN :
 + Gồm các chức năng INSERT, UPDATE, DELETE, SELECT
 + Search
 + Phân trang
 + Login, Logout
 + Kiểm duyệt
 + Phân Quyền
